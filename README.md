Snakeino
==================
Snakeino is imitation of Nokias snake game using Arduino, dot matrix display, joystick and buzzer.

### Hardware

- Arduino UNO R3
- Dot matrix display module MAX7219
- XY joystick module
- Buzzer module

### Libraries

- [LedControl](https://github.com/wayoda/LedControl)
