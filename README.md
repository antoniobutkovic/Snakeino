Snakeino
==================
Snakeino is imitation of Nokias snake game using arduino, dot matrix display and joystick.

### Hardware

- Arduino UNO R3
- Dot matrix display module MAX7219
- XY joystick module

### Libraries

- [LedControl](https://github.com/wayoda/LedControl)
